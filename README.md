# Latenly — AI-Powered WhatsApp Lead Analytics for Orthodontists

Latenly is a premium SaaS platform designed to help orthodontists identify hidden sales opportunities in their WhatsApp conversations using Artificial Intelligence.

## 🔮 Features

- **WhatsApp Integration**: Secure QR-based connection (read-only).
- **AI Opportunity Identification**: Automatically classifies chats to find patients ready to book appointments.
- **Revenue Projections**: Estimates the value of untapped leads in your WhatsApp.
- **Custom Messaging**: Generates personalized follow-up templates for each lead.
- **Premium Dashboard**: High-end dark mode interface with glassmorphism and real-time insights.

## 🛠 Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern Design System), JavaScript (ES6+).
- **Backend/Auth**: Supabase (Authentication & Database).
- **API**: Custom lead analysis engine.
- **Deployment**: Optimized for static hosting (GitHub Pages).

## 🔒 Security & Privacy

Latenly uses a read-only connection to analyze existing conversations. No data is modified, and no messages are sent on behalf of the user without their action.
